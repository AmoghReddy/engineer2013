Description

Virtual Bounty is the online technical treasure hunt of Engineer, where the divine 'Googling' skills  of the participants are put to test. Using search engines, the participants need to figure out the clues  which will take them all around the World Wide Web in order to get to the next more intriguing  level. This is a quest for the elusive treasure buried beneath billions of web pages.
